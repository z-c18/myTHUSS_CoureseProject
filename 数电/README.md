## 数电
#### 实验课
看上去很烦，实际上只要搞清楚要做什么就很简单了，基本上完成了需求就都是十分。
#### 理论课
没怎么听懂，平时作业基本就靠看习题解答写的，不放了。
#### 考试
讲的很多玄学的东西考试都不怎么考，考试题型基本上都可以在习题集里见到，建议考前看一遍习题解答熟悉题目位置，开卷考试的时候翻起来方便。
考试题量非常大，我记得期中卡诺图考了很多，需要花很长时间但占比不高，以致于时间过了一半的时候才做了30%.....建议不会就跳，做完最重要。
